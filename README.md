# Rusty Ollama

A native desktop app for [Ollama](https://ollama.ai) built with [Tauri](https://tauri.studio).

_üêâ Here be dragons! This project is in early stages of development_

## Install

Clone and at the root run

```shell
pnpm install
```

## Run

### Dev

```shell
pnpm tauri dev
```

### Build for production

```shell
pnpm tauri build
```
